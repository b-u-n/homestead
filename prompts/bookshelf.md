# Bookshelf Prompts

Use these prompts to generate new bookshelf images.

---

## Library Directory

A cozy wooden library directory board/sign in a warm, hand-crafted style. Framed with a simple wooden border, slightly aged. The board has a warm cream or parchment background. Small decorative elements like a tiny book icon or leaf in the corners. Soft, inviting colors matching a cozy library aesthetic. No text on the sign - leave it blank for the UI to overlay. Vertical orientation, taller than wide. Soft shadows, warm lighting.
