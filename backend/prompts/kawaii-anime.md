# Kawaii Anime Avatar

## Base Prompt
Kawaii anime chibi style illustration of a {{adjective}} {{adverb}} {{noun}}, cute big eyes, pastel colors, adorable expression, soft shading

## Description
Creates adorable kawaii-style anime avatars with chibi proportions and soft pastel colors.

## Variables
- `{{adjective}}` - Character personality trait
- `{{adverb}}` - Character behavior or style
- `{{noun}}` - Base creature or character type

## Style Notes
- Chibi proportions
- Large expressive eyes
- Pastel color palette
- Soft, cute aesthetic
