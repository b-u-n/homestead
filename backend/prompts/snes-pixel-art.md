# SNES Pixel Art Avatar

## Base Prompt
SNES video game pixel art 16 bit style character portrait of a {{adjective}} {{adverb}} {{noun}}

## Description
Creates retro 16-bit SNES-style pixel art avatars with vibrant colors and classic video game aesthetics.

## Variables
- `{{adjective}}` - Character personality trait
- `{{adverb}}` - Character behavior or style
- `{{noun}}` - Base creature or character type

## Style Notes
- 16-bit pixel art style
- SNES color palette
- Character portrait format
- Retro gaming aesthetic
