# Vaporwave Aesthetic Avatar

## Base Prompt
Vaporwave aesthetic synthwave neon portrait of a {{adjective}} {{adverb}} {{noun}}, neon colors, retro futuristic, 80s aesthetic, glowing outlines

## Description
Creates neon-soaked vaporwave style avatars with synthwave colors and retro-futuristic vibes.

## Variables
- `{{adjective}}` - Character personality trait
- `{{adverb}}` - Character behavior or style
- `{{noun}}` - Base creature or character type

## Style Notes
- Neon color palette (pink, cyan, purple)
- Glowing effects
- 80s retro-futuristic aesthetic
- Synthwave styling
